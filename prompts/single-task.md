# Single Task

Work on exactly one task. Finish it or stop.

1. Pick one task.
2. Complete it fully — code, tests, verification.
3. Commit.
4. Stop.

## Rules

- Do not start a second task in the same session.
- Do not "quickly fix" something unrelated while you're in the code. Note it for later.
- If the task is blocked, stop and say why. Do not switch to a different task to stay busy.
- If the task grows beyond what you expected, split it and finish only the first piece.
